# Portfolio - Almirco

Repositorio de proyectos de análisis de datos.

## Proyecto: Superstore Data Mart & Sales Analysis

Análisis de ventas usando un enfoque de Data Mart.

### Herramientas
- Archivo plano (CSV)
- SQL Server
- Visual Studio 2022
- Power BI

### Actividades realizadas
- Modelado de datos
- Consultas SQL para KPIs
- Dashboard interactivo en Power BI

### Resultados
- Análisis de ingresos, costos y ganancias
- Ventas por categoría, región y tiempo
